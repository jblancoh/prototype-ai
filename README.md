# BarrilitoDev Prototype AI

## Paso 1: Clonar el repositorio

```bash
git clone https://github.com/jblancoh/prototype-ai.git
```

```bash
cd prototype-ai
```

## Paso 2: Cambiar la rama a step indicado

```bash
git checkout step-2
```

## Paso 3: Instalar dependencias

```bash
npm install
```

## Paso 4: Iniciar el servidor

```bash
npm run dev
```
